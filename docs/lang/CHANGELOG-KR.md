## [1.1.0] – 2025-10-18

### ✨ 추가됨

- **스마트 단락 그룹화 엔진**
원래 Markdown 레이아웃을 유지하면서 여러 줄로 된 단락을 응집력 있는 블록으로 변환하는 지능형 단락 그룹화가 추가되었습니다.
→ 줄 정렬을 변경하지 않고도 보다 정확하고 상황에 맞는 번역을 생성합니다.

- **Diff 미리보기 모드(실험적)**
이제 사용자는 다음을 사용하여 출력을 저장하기 전에 번역 차이를 미리 볼 수 있습니다.
VS Code 출력 패널의 `auto-translate-readmes.previewDiff` 명령.

- **README 문자 세트 자동 감지**
플랫폼 간 일관성을 위해 소스 파일 인코딩(`UTF-8`, `UTF-16` 또는 `UTF-8-BOM`)을 자동으로 감지하고 보존합니다.

---

### 🔧 변경됨

- **인라인 마크다운 토크나이저 v3**
복잡한 Markdown 조합을 보호하기 위해 업데이트된 토크나이저:

- `**bold + _italic_ mix**`
- `[Click _here_](url)`
- `<span>` 또는 `<img>`과 같은 인라인 HTML 요소

- **코드 블록 일관성**
이제 코드 블록은 원래의 들여쓰기와 간격으로 완벽하게 보존됩니다(자르기나 추가 줄 바꿈 없음).

- **향상된 언어 전환기 형식**
다국어 전환기는 이제 각 대상 언어에 대한 구두점 및 간격 스타일을 자동으로 조정합니다(e.g., 일본어 전각 콜론, 콜론 앞의 프랑스어 공백).

- **보다 안정적인 Google API 통합**
이제 `translate.googleapis.com`이 속도 제한에 도달하거나 네트워크 오류가 발생하는 경우 보조 엔드포인트로의 자동 대체가 포함됩니다.

---

### 🐞 수정됨

- **굵게 처리(가장자리 케이스)**
`**Before:**` 또는 `**After (Absolute):**`이 `* *Before:**`로 분할되는 문제를 해결했습니다.

- **목록에서 줄바꿈하지 않는 공백**
목록 항목의 하이픈 `-`이 일부 언어(e.g., 프랑스어 또는 러시아어)의 텍스트와 직접 병합되는 문제가 방지되었습니다.

- **후행 줄 바꿈 유지**
이제 번역 출력은 소스 파일과 동일한 수의 빈 줄을 유지하며 아무것도 잘리지 않습니다.

- **이모지 안정성**
번역 중에 이모티콘이나 특수 문자가 삭제되는 인코딩 문제가 해결되었습니다.

- **교차 플랫폼 EOL 일관성**
소스 파일에 따라 일관된 줄 끝 문자(Windows의 경우 `\r\n`, Unix의 경우 `\n`)를 보장합니다.

---

### 🚀 개발자 노트

- 번역 로직은 이제 `translateBlock()`에서 비동기 일괄 처리로 모듈화되어 ~30% 더 빠른 번역을 제공합니다.
- 생성된 각 README에는 메타데이터 주석이 포함되어 있습니다.
`<!-- Auto-Translated v1.1.0 -->`
빌드 버전을 자동으로 추적합니다.
- **VS Code Marketplace v1.1 안정 릴리스** 준비가 완료되었습니다.

## [1.0.9] - 2025-10-17

### 변경됨

- **정밀 번역 엔진 (최종 개정판)**
원본 `README.md`의 _정확한_ 구조, 간격 및 개행 개수를 유지하기 위해 번역 프로세스를 완성했습니다.
- 번역된 모든 파일에서 동일한 레이아웃을 보장하는 줄바꿈 보존 논리(`split(/(\n)/)`)를 도입했습니다.
- 목록, 표, 코드 블록, 제목, 링크 및 이미지에 대한 번역을 안전하게 건너뛸 수 있도록 마크다운 인식이 개선되었습니다.
- 소스 파일과의 1:1 줄 매핑을 유지하기 위해 형식 정규화(e.g., 줄 바꿈 트리밍, 간격 정규식)를 제거했습니다.
- 혼합 콘텐츠 줄(인라인 코드, 이모티콘 또는 목록 표시자가 있는 텍스트)에 대한 처리가 향상되었습니다.
- 보장된 출력은 일관된 UTF-8 인코딩 및 보존된 EOL 형식(Windows `\r\n` 또는 Unix `\n`)을 사용하여 작성됩니다.

### 수정됨

- **정확한 마크다운 충실도** – 이제 번역된 파일이 시각적으로나 구조적으로 소스 `README.md`과 정확히 일치합니다.
- **들여쓰기 복원** – 하위 목록 또는 중첩된 목록 항목이 번역 후 들여쓰기가 손실되는 문제가 해결되었습니다.
- **빈 줄 보존** – 단락과 목록 사이에 사라지는 빈 줄을 수정했습니다.
- **굵게 & 인라인 코드 처리** – 줄 시작 부분의 `**bold**` 및 `inline code` 형식이 잘못된 문제를 해결했습니다.
- **테이블 줄 무결성** – 라틴어가 아닌 언어(e.g., 중국어, 한국어)의 복잡한 Markdown 테이블에서 손상된 구분선을 수정했습니다.

## [1.0.7] - 2025-10-17

### 추가됨

- **사용자 피드백**: 이제 "생성" 버튼이 회전하는 아이콘으로 표시되고 번역 중에는 비활성화됩니다.

### 수정됨

- 종속성 누락으로 인한 버그 게시.
- 누락된 공백, 깨진 목록 표시, 잃어버린 이모티콘 등을 포함한 서식 오류입니다.
- 언어 전환기 헤더가 중복되었습니다.
- 빌드 성공을 위해 누락된 유형 종속성을 추가했습니다.
